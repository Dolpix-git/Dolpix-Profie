Rope Demo
Experiment

This was testing the concept of a rope system that uses dynamic particles. The points of collision custom generate particles to then act like a particle rope. Pro's include better performance due to less particles but less physically accurate ropes.

IK Rig
Experiment

The goal here was to use default animations to power a IK rig, allowing for me to dynamically change animations in run time.
